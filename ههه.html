أذكار

✅ أذكار الصباح

أَصْـبَحْنا وَأَصْـبَحَ المُـلْكُ للهِ رَبِّ العَـالَميـن، اللّـهُمَّ إِنِّي أَسْأَلُكَ خَـيرَ هَـذا الْيَومِ، فَتْحَهُ، نَصْرَهُ، نُورَهُ، بَرَكَتَهُ، وَهُدَاهُ، وَأَعوذُ بِكَ مِنْ شَرِّ ما فيهِ وَشَرِّ ما بَعْدَهُ.

✅ أذكار المساء

أَمْسَينا وَأَمْسى المُـلْكُ للهِ رَبِّ العَـالَميـن، اللّـهُمَّ إِنِّي أَسْأَلُكَ خَـيرَ هَـذا اللَيلِ، فَتْحَهُ، نَصْرَهُ، نُورَهُ، بَرَكَتَهُ، وَهُدَاهُ، وَأَعوذُ بِكَ مِنْ شَرِّ ما فيهِ وَشَرِّ ما بَعْدَهُ.

0

✅ أذكار النوم

بِاسْمِكَ رَبِّي وَضَعْتُ جَنْبِي، وَبِكَ أَرْفَعُهُ، إِن أَمْسَكْتَ نَفْسِي فَارْحَمْهَا، وَإِن أَرْسَلْتَهَا فَاحْفَظْهَا بِمَا تَحْفَظُ بِهِ عِبَادَكَ الصَّالِحِينَ.

0

✅ أذكار بعد الصلاة

اللّهمَ أَنْتَ السَلامُ وَمِنْكَ السَلامُ، تَبَارَكْتَ ذا الجَلالِ وَالإِكْرامِ.

0

✅ أذكار الخلاء

اللّهمَ إِنِّي أَعوذُ بِكَ مِنَ الْخُبْثِ وَالْخَبائِثِ.

0

✅ أذكار الأكل والشرب

بِسْمِ اللَّهِ.

0

✅ أذكار السفر

اللّهمَ إِنَّا نَسْأَلُكَ في سَفَرِنا هذا البِرَّ وَالتَقْوى، وَمِنَ العَمَلِ ما تَرْضى.

0

✅ أذكار الاستيقاظ من النوم

الْحَمْدُ لِلَّهِ الَّذِي أَحْيَانا بَعْدَ ما أَمَاتَنا وَإِلَيْهِ النُّشُورُ.

0

✅ أذكار دخول المنزل والخروج منه

بِسْمِ اللَّهِ تَرَكْنا وَبِسْمِ اللَّهِ دَخَلْنا، وَعَلَى رِزْقِ رَبِّنا رَبِقَونَ.

0

✅ أذكار المسجد

اللّهمَ افْتَحْ عَلَيَّ خَيْرَ مَساجِدِكَ وَأَدْخِلْنِيَ بِرَحْمَتِكَ.

0

✅ أذكار الحج والعمرة

لَبَّيْكَ اللّهمُ لَبَّيْكَ، لَبَّيْكَ لا شَرِيكَ لَكَ لَبَّيْكَ، إِنَّ الْحَمْدَ وَالنِعْمَةَ لَكَ وَالْمُلْكَ لا شَرِيكَ لَكَ.

0

✅ أذكار المرض والشفاء

أَسْأَلُ اللَّهَ الْعَظِيمَ رَبَّ الْعَرْشِ الْعَظِيمِ أَنْ يَشْفِيَكَ.

0

✅ أذكار الاستخارة

اللّهمَ إِنِّي أَسْتَخِيرُكَ بِعِلْمِكَ، وَأَسْتَقْدِرُكَ بِقُدْرَتِكَ، وَأَسْأَلُكَ مِنْ فَضْلِكَ الْعَظِيمِ.

0

✅ أذكار الكرب والهم

اللّهمَ إِنِّي أَعوذُ بِكَ مِنَ الْهَمِّ وَالْحَزَنِ، وَأَعوذُ بِكَ مِنَ الْعَجْزِ وَالْكَسَلِ.

0
